# 동기부여 시스템 적용 안내

## 무엇이 바뀌었나요

1. **gamify.js (신규 파일)** — 캐럿(크레딧)·닉네임·리그·배지·스트릭을 담당하는 공통 모듈입니다.
   서버·DB 없이 브라우저 `localStorage`에만 저장됩니다. 다른 4개 HTML 파일이 이 파일 하나를
   공유해서 불러 씁니다.

2. **문장 카드 맞추기 / 단어 짝맞추기 / 말하기 연습 / VOCA 트레이닝** (각 html 파일)
   - `<body>` 바로 다음 줄에 `<script src="gamify.js"></script>` 한 줄이 추가되었습니다.
     → 이 한 줄만으로 화면 맨 위에 "닉네임 · 리그 · 캐럿 · 스트릭"이 표시되는 얇은 바가 생깁니다.
   - 각 게임이 세트를 끝내는 지점(기존 `finishGame()` / `showDone()` 함수 안)에
     `Gamify.reportResult(...)` 호출이 한 줄~여러 줄 추가되었습니다. 게임 자체의 로직·화면·통계는
     전혀 건드리지 않았고, 결과가 나온 시점에 "보고만" 하도록 최소한으로 연결했습니다.

3. **index.html**
   - 상단에 "내 현황" 대시보드가 추가되어 닉네임·리그·캐럿·스트릭·배지 개수를 한눈에 보여줍니다.
   - 그동안 목록에 없던 **VOCA 트레이닝 카드**를 4번째로 추가했습니다(기존엔 3개 게임만 연결되어 있었어요).

## 캐럿 획득 규칙 (요약)

| 게임 | 완료 기본 캐럿 | 정확도 보너스 |
|---|---|---|
| 문장 카드 맞추기 | 10 | 90%+ → +10 / 70~89% → +5 |
| 단어 짝맞추기 | 10 | 90%+ → +10 / 70~89% → +5 |
| 말하기 연습 | 15 | 90%+ → +10 / 70~89% → +5 |
| VOCA 트레이닝(각 모듈) | 10 | 90%+ → +10 / 70~89% → +5 |

- 당일 첫 학습 +5, 연속 3일 +20 / 7일 +50 / 30일 +200
- 일일 상한 300 캐럿, 15분 내 같은 게임 재플레이 시 기본 캐럿 절반
- 리그: 🥉브론즈(0) → 🥈실버(1,000) → 🥇골드(3,000) → 💎사파이어(7,000) → 👑다이아몬드(15,000)
- 리그 승급 시 리워드 신청 코드가 자동 생성되며, 보호자가 확인 후 리워드를 지급하는 방식입니다.

## 적용(배포) 방법

기존 GitHub 저장소에 이 폴더의 파일들을 **그대로 덮어쓰기**만 하면 됩니다.

```
git add gamify.js index.html sentence_card_game.html word_match_game.html speaking_practice_game.html voca_trainer.html
git commit -m "동기부여 시스템(캐럿/닉네임/리그/배지) 적용"
git push
```

manifest.json, service-worker.js, 아이콘 파일들은 원본과 동일하므로 바꾸지 않아도 됩니다
(비교를 위해 이 폴더에도 함께 넣어두었습니다).

## 배포 전 로컬 확인 방법

1. 이 폴더를 통째로 다운로드
2. `index.html`을 더블클릭해서 브라우저로 열기 (또는 `python3 -m http.server`로 로컬 서버 실행)
3. 처음 열리면 닉네임 입력 팝업이 뜹니다 → 아무 이름이나 입력
4. 아무 게임이나 한 세트 완료 → 우측 상단에 "+N 캐럿 획득!" 알림이 뜨는지 확인
5. index.html로 돌아가서 대시보드에 캐럿·스트릭이 반영되는지 확인

## 주의할 점

- 데이터는 **이 브라우저(이 기기)에만** 저장됩니다. 다른 기기·다른 브라우저로 접속하면 이어지지
  않으니, 필요하면 프로필 전환 메뉴의 "내 기록 내보내기/가져오기"로 옮겨주세요.
- 리그 강등은 없습니다. 스트릭이 끊겨도 등급은 유지됩니다(자녀 대상 서비스 특성을 고려한 설계).
- 기존 게임의 플레이 로직·UI·데이터 입력 방식(텍스트/엑셀/깃허브 불러오기)은 전혀 바뀌지 않았습니다.
